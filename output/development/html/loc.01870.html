Brooklyn
Dec 26 1872



Dear friend and Father

[ as it] is snowi [ here] tonight [ I?] was [ looking] of some letter [ I came] acrost one of [ your] letter and [ as] it [ some time] cince I have [ not seen or heard] from you [ I thought] I would [ write a few lines] to [ you. I am as] well as [ can be] and am [ getting very] Stout but [ how do] you do and [ are] you well and how [ do] you get [ along] this cold [ weather it has] been [ very Cold here] I am now [ employed] at the Contentatual Bank Note Co and I get alon very nicely do much better than I did with the [ wood] Co

Last [ Sunday we] had very [ large fire in] Brooklyn [ Dr. Talmage] Church [ in Schwirharn St.] was [ destroyed and] some [ very large fire in New York, by] Barnum[ 's Museum, and] a larg [ Printing Co] in Center [ St but I ] supose [ you have heard] of it [ by] this now I will [ close] with love

[ I remain] your

John M Rogers

Correspondent:
John (Jack) M. Rogers was a Brooklyn driver with whom Whitman presumably had a loving relationship. Whitman first met him in Brooklyn on September 21, 1870. For more on this, see Charley Shively (ed) Calamus Lovers. Walt Whitman's Working Class Camerados (San Francisco: Gay Sunshine Press, 1987, pp. 122-135).
